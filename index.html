intro

Mixing up javascript generated panels - A simple html 5 panel management library

The concept with this library is simple - you start with no html. So it needs no selectors or DOM queries or DOM parsers.

Every thing is a panel (or widget or layout) and a panel has a node which is a reference to the dom element - starting with a pseudo root panel which simply binds to document.body.
